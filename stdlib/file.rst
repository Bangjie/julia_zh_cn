
.. currentmodule:: Base

文件系统
--------

.. function:: isdir(s) -> Bool

   如果 ``s`` 是文件夹，则返回 ``true`` ；否则返回 ``false`` 。

.. function:: isfifo(s) -> Bool

   如果 ``s`` 是 FIFO ，则返回 ``true`` ；否则返回 ``false`` 。

.. function:: isfile(s) -> Bool

   如果 ``s`` 是文件，则返回 ``true`` ；否则返回 ``false`` 。

.. function:: islink(s) -> Bool

   如果 ``s`` 是符号链接，则返回 ``true`` ；否则返回 ``false`` 。
